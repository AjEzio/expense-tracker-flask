# Expense Tracker using Flask

A small **web-based** expense tracker written in Python. You can add expenses, view them as rows, and see a total—all stored in a simple CSV file.


## Home Page

After you run the program, you’ll see a home page with buttons for:

| What it does |
|----------------|
| Add an expense (date, amount, category, note) |
| View saved expenses (printed from the CSV) |
| Show total expense |

**Press the buttton for the function stated**.

## Where your data is saved

Expenses are stored in:

`expenses.csv`

The first time you run the app, that file is created automatically with a header row (`date`, `amount`, `category`, `note`).
