# Expense Tracker using Flask

A small **web-based** expense tracker written in Python. You can add expenses, view them as rows, and see a total—all stored in a database.


## Home Page

After you run the program, you’ll see a home page with buttons for:

| What it does |
|----------------|
| Add an expense (date, amount, category, note) |
| View saved expenses (printed from the database) |
| Show total expense |

**Press the buttton for the function stated**.

## Where your data is saved

Expenses are stored in a able expenses in app.db


The first time you add an expense, that database is created automatically with a header row (`date`, `amount`, `category`, `note`).
